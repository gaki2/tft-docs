# 빌지워터덱

빌지워터덱은 1등 포텐이 있는 덱 입니다.
빌지워터 유닛이 공격한 적 챔피언에게 폭탄이 떨어져 데미지를 입히는데, 시너지가 높아질수록
폭탄의 데미지가 굉장히 강해집니다.


## 언제 빌지워터 덱을 해야하나요?
> - 초반에 미스포춘, 노틸러스가 나왔을때
> - 빌지워터 상징을 2-1 증강에서 획득했을때

### 후반 단계 운영법 (4-2 이후)

#### 1. 빌지워터 상징이 없을 경우
빌지워터 상징이 없다면 5 빌지워터 까지만 시너지를 맞추고, 나머지 자리는 다른 시너지를 맞추는게 좋습니다.

만약 갱플랭크가 나와서 7빌지워터를 할 수 있더라도, 왠만한 상황에서는 5빌지워터로 유지하는 것이 좋습니다.

왜냐하면 빌지워터 유닛에는 1원짜리 유닛인 <ChampionBadge season={'season_9b'} championName={'그레이브즈'} /> <ChampionBadge season={'season_9b'} championName={'일라오이'} /> 가 있는데 1원짜리 유닛은 후반으로 갈수록 썩기 때문입니다.

따라서 7빌지워터를 하는 것 보다 5빌지워터를 하고 나머지 두 자리에는 4원이나 5원짜리 고코스트의 유닛을 넣어주는것이 좋습니다.
